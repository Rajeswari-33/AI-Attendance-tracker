# AI Attendance Tracker

# Project Overview

The AI Attendance Tracker is a web-based application developed to modernize and automate the traditional attendance tracking process. This system leverages intelligent analysis to monitor student attendance patterns, provide timely alerts, and promote academic discipline through automated motivational engagement.

# Objectives
- Automate and simplify attendance management
- Reduce manual errors and administrative workload
- Identify students at risk due to low attendance
- Encourage consistency through motivational feedback
- Provide data-driven insights using AI techniques

# Features
- Tracks student attendance automatically
- Alerts students if attendance is below 70%
- Sends motivational messages for 100% attendance
- Shows attendance reports using charts
- Student dashboard to view progress
- Faculty dashboard to manage attendance
- Admin dashboard to manage system
